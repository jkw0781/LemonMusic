본 서비스의 개발 목적

본 서비스는 '단순한 음악 차트 사이트'를 목적으로 개발되었으며, 국내·해외 음원 차트 사이트를 참고하여 디자인 되었습니다.
본 서비스는 아래의 사이트들의 디자인에서 장점만을 뽑아 제작되었습니다.


· TIDAL (https://tidal.com/) - 메뉴바 디자인

· billboard (https://www.billboard.com/) - 차트의 베스트3 요소 모티브

· Melon (https://www.melon.com/) - 프로젝트명, 서비스 BI 모티브

· YouTube Music (hhttps://music.youtube.com) - 차트 UI / 검색 UI / 탐색 UI 참고


프로젝트 기획 초안에서는 '차트' 기능만을 기획 및 구현하였고, 차트 구현이 완료된 후 위 사이트와 구현된 차트 UI를 참고하여 음원 검색, 플레이리스트 탐색 기능을 추가로 구현하여 서비스를 완성했습니다.

사용된 기술

※ 본 서비스는 FastAPI에 의해 구동 및 호스팅됩니다.

1. 실시간 차트 / 노래, 아티스트 검색 결과 / 플레이리스트

본 서비스는 Python으로 구축된 백엔드 서버에서 정보 표시에 필요한 데이터를 전달받습니다. 각 서비스의 동작 방식은 아래 이미지로 확인 가능합니다.

2. 매인 페이지의 fullPage 레이아웃

본 서비스의 프론트 파트는 html / css (FontAwesome) / JS (jQuery) 를 사용하여 제작되었으며, 매인 페이지의 fullPage 레이아웃 역시 외부 라이브러리를 사용하지 않은 Javascript로 작업되었습니다.

본 서비스에 대하여

1. 이용 약관

멤버 기능 업데이트로 아래 이용 약관이 추가되었습니다.

01. 개인정보처리방침
- 여기에서 본 서비스의 개인정보처리방침 내용을 확인 할 수 있습니다.

02. 서비스 이용 약관
- 여기에서 본 서비스의 서비스 이용 약관 내용을 확인 할 수 있습니다.

2. API

본 서비스에서는 비공식 Python 모듈인 ytmusicapi가 사용됩니다.

3. 개발 환경

본 서비스는 아래 환경에서 개발 및 호스팅됩니다.

       - Python 3.11.x or higher
       - Javascript
       - FastAPI

4. 기타 서비스 정보

01. 디자인 오브젝트
- 본 사이트 내에서 사용된 모든 이미지에 대한 저작권은 개발자에게 있습니다. 단, 앨범 아이캐치 이미지의 경우 음원 권리사에 저작권이 있음을 알려드립니다.

02. 사이트 목적
- 본 서비스 및 사이트는 오직 개발자의 포트폴리오로만 사용됩니다.

본 사이트는 비공식 API를 사용하며, 오직 포트폴리오의 목적으로만 개발 및 사용됩니다.
사이트 디자인에 사용된 모든 이미지 (단, 곡 정보에 포함된 앨범 아이캐치는 제외)의 저작권은 사이트 개발자에게 있습니다.
